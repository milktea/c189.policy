c189.policy Installation
------------------------

To install c189.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``c189.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        c189.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
